# Markdown Writing Guide for Authors

## Basic Formatting

### Bold Text

```markdown
**This text will be bold**
```

Result: **This text will be bold**

### Italic Text

```markdown
_This text will be italic_
```

Result: _This text will be italic_

### Bold and Italic

```markdown
**_This text will be both bold and italic_**
```

Result: **_This text will be both bold and italic_**

---

## Headings

```markdown
# Chapter Title (Heading 1)

## Section Title (Heading 2)

### Subsection Title (Heading 3)

#### Minor Heading (Heading 4)
```

---

## Lists

### Unordered List

```markdown
- First item
- Second item
- Third item
  - Nested item
  - Another nested item
```

Result:

- First item
- Second item
- Third item
  - Nested item
  - Another nested item

### Ordered List

```markdown
1. First step
2. Second step
3. Third step
   1. Sub-step
   2. Another sub-step
```

Result:

1. First step
2. Second step
3. Third step
   1. Sub-step
   2. Another sub-step

---

## Quotes and Dialogue

### Blockquotes

Perfect for internal thoughts or special narration:

```markdown
> "This is a character's thought," he mused silently.

> The ancient text read: "Seek ye the truth beyond the veil, Where shadows dance
> and light grows pale."
```

Result:

> "This is a character's thought," he mused silently.

---

## Separators

Use horizontal rules to separate scenes or sections:

```markdown
The battle raged on through the night.

---

The next morning, silence fell over the battlefield.
```

---

## Links

### External Links

```markdown
For more information, visit [our website](https://example.com).
```

### Reference-style Links

```markdown
This is a [reference link][1] to another page.

[1]: https://example.com "Optional Title"
```

Useful for author notes or todo items:

```markdown

- [x] Draft first scene

- [ ] Review and edit
```

## Writing Tips

### Scene Breaks

Use horizontal rules to indicate scene changes:

```markdown
The hero walked away into the sunset.

---

Meanwhile, in the villain's lair...
```

### Emphasis Combinations

```markdown
He whispered, "_Please_, **you must believe me**!"

The notification appeared: **[Quest Complete!]** _Rewards: 5000 EXP, Rare Item
Box_
```

### Dialogue Formatting

```markdown
"What do you mean?" Sarah asked.

"I mean," John replied, his voice barely above a whisper, "that **everything**
has changed."

She stared at him in disbelief. _This can't be happening_, she thought.
```

## Best Practices

1. **Keep paragraphs short** - Use double line breaks for readability
2. **Use headings sparingly** - Only for major scene or time changes
3. **Be consistent** - Pick a style for dialogue and stick with it
4. **Use emphasis wisely** - Don't overuse bold or italic text
5. **Preview often** - Check how your chapter looks before publishing

---

## Keyboard Shortcuts in Editor

- Use the **toolbar buttons** for quick formatting
- Select text first, then click formatting to wrap selected text
- The **Split View** shows live preview as you type
- **Character counter** helps track chapter length
- **Word counter** useful for pacing and targets
